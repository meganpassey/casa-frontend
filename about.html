About CaSa

CaSa is an app that uses the barometer function of a smartphone to accurately map elevation.

How does it work?

1. Get maps

use the app to download a base map on to the phone

2. Collect data

smartphones to collect data on the ground

3. Send data

collected data is uploaded on to a web server

4. Download map data

downloaded data can be used to make maps

Why do we need it?

The app was developed to provide detailed information to site planners in humanitarian emergencies.
A lack of accurate elevation data means that camps for displaced people are sometimes vulnerable to flooding because of subtle changes in elevation that cannot be detected from satellite imagery.
CaSa is easy to use and can be deployed as soon as a new site is identified.
CaSa can also collect data to help prepare an existing camp for the rainy season.
By measuring elevation data tent by tent, the app can be used to identify shelters vulnerable to flooding so that they can be protected or relocated.
